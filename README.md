# Purushottam Barai - Portfolio

A personal portfolio website showcasing my projects, skills, and blog posts. Built with a focus on clean design, responsiveness, and performance.

## Features

- Responsive Design: Adapts seamlessly to desktop, tablet, and mobile devices.
- Dynamic Sliders: Custom vanilla JavaScript implementations for project highlights and blog posts.
- Modern UI/UX: Dark-themed interface with smooth transitions and glassmorphism elements.
- Static Architecture: Fully standalone frontend utilizing only HTML, CSS, and JavaScript.

## Tech Stack

- HTML5
- CSS3 (Vanilla)
- JavaScript (Vanilla)

## Getting Started

To view the website locally, simply download or clone the project and open the `index.html` file in any modern web browser. No server setup, build tools, or external dependencies are required.

## Project Structure

- `index.html`: The main markup file containing the layout and content.
- `index.css`: All styling, including animations and responsive media queries.
- `Images/`: Directory containing project screenshots and background assets.
- `Resume`: Downloadable resume file.
